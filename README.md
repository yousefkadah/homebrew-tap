# yousefkadah/homebrew-tap

Homebrew formulae.

## groundrules

```bash
brew install yousefkadah/tap/groundrules
```

[Groundrules](https://github.com/yousefkadah/groundrules) — one source of truth for AI coding agents:
detect your stack, scaffold `.ai/`, and generate every agent's rules file (AGENTS.md, CLAUDE.md, Cursor,
Copilot, Gemini). This installs the **node-free Rust binary**.
